# ⚙️ Best Practice

## 1. Nicht alle Codes speichern
→ nur relevante Subsets

## 2. Codes gruppieren
→ bessere UI / Filter

## 3. Codes versionierbar halten
→ ICD ändert sich jährlich

## 4. Erweiterbar bleiben
→ später kompletter Import möglich
